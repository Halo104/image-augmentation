Random Image Augmentation

This website is designed to conduct one of four different image augmentation techniques to your choice of image. The only requirement is to have your image accessible through a URL and follow the instructions below.

Enter URL

Please enter an image URL to send an image and get the app to work, if you are hosting it locally then follow the first set of instructions. If it has been hosted on azure then follow the second set of instructions.

Hosted locally:
http://0.0.0.0:8000/?url=<image_url> or http://localhost:8000/?url=<image_url>

for example:

http://0.0.0.0:8000/?url=https://secureservercdn.net/160.153.138.219/k58.7f8.myftpupload.com/wp-content/uploads/2020/11/Object-and-material-INDUSTRIAL.png
or
http://localhost:8000/?url=https://secureservercdn.net/160.153.138.219/k58.7f8.myftpupload.com/wp-content/uploads/2020/11/Object-and-material-INDUSTRIAL.png

Hosted on Azure:
https://recycleye-image-augmentation.azurewebsites.net/?url=<image_url>

for example:

https://recycleye-image-augmentation.azurewebsites.net/?url=https://secureservercdn.net/160.153.138.219/k58.7f8.myftpupload.com/wp-content/uploads/2020/11/Object-and-material-INDUSTRIAL.png
